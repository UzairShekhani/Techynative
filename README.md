### Run Locally

To run **Desinic** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/desinic.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/desinic.git
```
